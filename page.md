# 페이지 기획

P01. 온보딩 / 스플래시

|항목|내용|
|---|---|
|목적|서비스 첫인상 전달 + 회원가입 유도|
|주요 구성|앱 로고 + 슬로건 / 서비스 핵심 3가지 소개 (슬라이드) / [시작하기] [로그인] 버튼|

## P02. 회원가입 & 인증

|항목|내용|
|---|---|
|목적|신뢰 기반 사용자 유입|
|주요 구성|이메일 입력 → 인증 코드 발송 / 사용자 유형 선택: 대학생·직장인 / 학교 or 회사 이메일 인증 / |닉네임·프로필 사진 설정 / 인증 완료 시 뱃지 부여 안내

## P03. 홈 (메인 피드)

|항목|내용|
|---|---|
|목적|핵심 기능 진입점 + 주변 팟 탐색|
|상단|출발지 / 도착지 / 시간 검색바 (고정)|
|중단|지도 뷰 — 내 위치 기반 팟 핀 표시|
|하단|리스트 뷰 — 팟 카드 목록 (거리 가까운 순)|
|팟 카드|출발지→목적지, 출발 시간, 현재/최대 인원, 인증 여부|

## P04. 팟 상세 페이지

|항목|내용|
|---|---|
|목적|팟 정보 확인 후 참여 결정|
|주요 구성|지도: 출발지~도착지 경로 시각화 / 팟 정보: 시간·인원·예상 비용(1/n)·픽업 포인트 / 참여자 목록 |(닉네임 + 인증 뱃지) / [팟 참여하기] 버튼 → 채팅방 입장

## P05. 팟 생성 페이지

|항목|내용|
|---|---|
|목적|새로운 팟 모집|
|주요 구성|출발지/도착지 검색 (지도 핀 설정) / 출발 시간 선택 (날짜+시간 picker) / 최대 인원 설정 |2~4명 / 이동 수단: 택시·개인 차량 / 한 줄 메모 입력 / [모집 시작] 버튼

## P06. 채팅방

|항목|내용|
|---|---|
|목적|팟 참여자 간 실시간 소통|
|주요 구성|상단 고정 배너: 팟 요약 정보 / 실시간 메시지 + 타임스탬프 / 📍 위치 공유 / ✅ 준비 완료 상태 |표시 / 이동 완료 후 매너 평가 팝업

## P07. 검색 & 필터

|항목|내용|
|---|---|
|목적|원하는 팟 정밀 탐색|
|주요 구성|출발지/도착지/시간대 입력 / 필터: 인증 사용자만 보기·이동 수단·거리 반경(300m~1km) / 지도 뷰 |↔ 리스트 뷰 전환

## P08. 마이페이지

|항목|내용|
|---|---|
|목적|개인 정보 관리 + 이용 내역|
|주요 구성|프로필: 사진·닉네임·인증 뱃지 / 매너 점수 (평균 평가 기반) / 탑승 내역: 지난 팟 목록 / |즐겨찾는 경로 저장 / 알림 설정 / 로그아웃
